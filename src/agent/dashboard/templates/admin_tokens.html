Hướng dẫn sử dụng

Sau khi tạo token, dùng header X-API-Token khi gọi POST /trigger:

curl -X POST http://localhost:8000/trigger \
  -H "Content-Type: application/json" \
  -H "X-API-Token: <token>" \
  -d '{"service":"payment-gateway","scenario":"scenario1","time_window":"14:00-15:00"}'

Để tắt auth (dev only): đặt ALLOW_ANON_TRIGGER=true trong .env.
